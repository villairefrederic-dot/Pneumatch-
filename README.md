# PneuMatch
**Application de recommandation intelligente de pneus**

## Description
PneuMatch aide les automobilistes à trouver les meilleurs pneus en fonction de leur véhicule, usage, et budget.

## Fonctionnalités
- Classement par notes (freinage, usure, bruit).
- Filtres : gabarit, saison, type de conduite.
- Affichage des 3 pneus les plus adaptés.
